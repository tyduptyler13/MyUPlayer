MyUPlayer
====
A new web equalizer writen for HTML5 using JQuery, Three.js, and WebAudio

A working version is available [here](http://hgtti.com/music).

Features
----
* Equalizer bars
* Color based on frequency
* Audio Element controls
* Sample Smoothing
